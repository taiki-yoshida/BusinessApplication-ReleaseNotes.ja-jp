#  <a name="facility-scheduling"></a>設備のスケジューリング

[!include[field-service banner](../../../includes/field-service.md)]

顧客所在地の代わりにサービス プロバイダーの場所でサービスを提供します。

* シナリオの例
    * 物理スペースを予約します
        * イベントまたはパーティのために部屋を予約します
        * 演習クラスのために部屋を予約します
        * 修理工場でベイを予約します
        * ボートを予約します
    * 施設で人と会う約束をします
        * Microsoft の小売店でラップトップを修理します
        * 銀行での資産管理コンサルテーション
        * フォーカス ルームでの個人指導
