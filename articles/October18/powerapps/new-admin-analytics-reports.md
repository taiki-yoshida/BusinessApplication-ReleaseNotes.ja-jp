# <a name="new-admin-analytics-reports"></a>新しい管理分析レポート


[!include[banner](../../includes/banner.md)]

アプリ用 Common Data Service の管理者は、環境に組織の分析情報ソリューションをインストールする必要がなくなりました。組織の分析情報レポートはすべて管理センターから (既定で) 使用できます。
